## React redux cart

This is a repository of Shopping cart app built using react and redux.

## Project Setup:

1. Install node modules :

```bash
 npm i
```
2. First run the development :

```bash
 npm run dev
```

3. Start the server :

```bash
 npm run dev
```

4. Open the browser :
 
```bash
 http://localhost:5173
```
